* Install python version 3.8 to your MOS/Linux/Windows operating system.
* Install Anaconda Navigator and create your own environment or use the default base(root)     environment to run your code.
* Download Health Care Dataset from the code section.
* Download provided PNG/JPG files.And Save these files to the exact location where you saved your dataset.
* Download the source code from the code section from git repository.
* Launch the jupyter notebook from the anaconda navigator.
* Import required python libraries for this stroke prediction system. for example, NumPy, matplotlib, seaborn, etc.
* Execute each block code in jupyter notebook or use python IDE to compile or run your code.
